How to install Atom on your machine



1. Get latest Binaries

On Windows 10.0 or MacOS you can simply download binaries from the official atom site. Fire up a webbrowser of your choice and paste the link: https://atom.io/download/windows_x654

2. Run the Binary
After downloading the atom editor installer-file, go into the installation location and run the installer. Now wait a moment until atom unpacks all of it's chromium sources. Then, run the atom editor and customize it. You might now wan't to go for a coffee now :D
